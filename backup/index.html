geogebra使用备注

from github(gitme)

  1. 手工画图, 然后导出pgf/tikz, 再放到standalone.tex中生成pdf插入.tex中去:
    1. 把坐标系放大, 正常情况下\(4\times 4\)正好;
    2. 图画好后, 将图放在第一象限, 最好贴着坐标轴, 理由在下一步;
    3. 在导出pgf/tizk代码时, 在对话框中填入\(x_{\min},y_{\min}, x_{\max},y_{\max}\), 生成pgf/tikz代码;
    4. documentclass选择为standalone, 拷贝代码, xelatex生成pdf.

  2. 高级选项中的任何修改, 都要返回上级菜单, 选择"保存设置".

  3. 新增文件, 写好文件, 然后上传. 只有上传是git的事情, 其他都不是.

git add git.md git commit -m "注解说明" git push origin master ```
